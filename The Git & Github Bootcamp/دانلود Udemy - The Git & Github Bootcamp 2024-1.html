Udemy – The Git & Github Bootcamp 2024-1

در تاریخ: 1403/05/11 4:36:43
در: آموزش تصویری
نظرات (4)

توضیحات

The Git & Github Bootcamp، یک دوره آموزش گیت و گیت هاب میباشد که بوسیله آن نکات مختلفی در موارد گوناگونی مثل rebasing, squashing, stashing, reflogs, blobs, trees را یاد خواهید گرفت. Git یک ابزار حیاتی برای کار در حیطه های مربوط به کدنویسی میباشد که میتواند از علوم داده تا توسعه بازی یا یادگیری ماشین را شامل شود. این دوره همه آنچه شما نیاز دارید بدانید تا بتوانید Git و Github را شروع کنید شامل میشود. مباحث این دوره در چهار بخش اصلی شامل مبانی گیت، سطح پیشرفته گیت، Github و Collaboration و در نهایت قسمت آخر که در آن دستورات سودمند گیت و مباحث پیشرفته تر آموزش داده میشود گنجانیده شده اند.

آنچه در دوره The Git & Github Bootcamp یاد خواهید گرفت:

  • درک چگونگی کارکرد Git
  • توضیح تفاوت بین اشیا Git: شامل trees, blobs, commits, and annotated tags
  • کسب مهارت در گردش کار Git: اضافه و ذخیره کردن
  • کار با شاخه های Git
  • انجام ادغام Git و رفع کردن تعارض های ادغام
  • استفاده از Git diff برای نشان دادن تغییرات در طول زمان
  • کسب مهارت در Git stashing
  • خنثی کردن تغییرات با استفاده از git restore, git revert و git reset
  • کار با مخازن محلی و ریموت
  • کسب مهارت در گردش کار Collaboration
  • Squash پروژه، پاک سازی و نوشتن مجدد تاریخچه با استفاده از rebase تعاملی
  • بازیافت پروژه های گم شده با استفاده از git reflogs
  • نوشتن نام های مستعار Git سفارشی و قدرتمند
  • علامت زدن release و نسخه ها با استفاده از تگ های Git
  • میزبانی سایت های ایستا با استفاده از صفحات Github
  • ساخت markdown README
  • اشتراک گذاری کد ها با استفاده از Github Gists
